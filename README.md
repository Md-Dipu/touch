## Touch: Command-line app for files creator

This is a command-line application. That creates one or more files. Will not create existing files. This simple CLI app is created in C language.

> The purpose of this program is learning. Inspired by Linux CLI Command 'touch', which is used to create files, I tried to make this console app.

#### _Command:_

```
touch [-help | /h] <filename>
```

- filename: Name of file/files to create.
- -help, /h: Get user guide documentation.

To create more than one file attach filename using (\<file1> \<file2> \<file3>
... \<fileN>) format.
